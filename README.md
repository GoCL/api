# GoCL_API

## 介绍
* 基于 Gin 的 API 开发框架
* go 版本 >= 1.6
* 目录结构
    ```
    ├── config                  //配置文件
    ├── controller              //逻辑层
    ├── doc                     //文档
    ├── library                 //拓展库
    │   ├── common
    │   ├── error
    │   └── system
    ├── middleware              //中间件
    ├── model                   //模型层
    ├── public                  //静态资源
    ├── router                  //路由
    └── vendor                  //第三方依赖
    ```
